AI-first, secure by design

Vanrot ships a machine-readable manifest your agent reads directly — and a runtime with no eval, scoped styles, and zero supply-chain surface.

⌥

Agent manifest

Structured docs/ai knowledge. Agents understand Vanrot without guessing.

⊘

No eval

Templates compile to plain JS. CSP-friendly, nothing evaluated at runtime.

▣

Zero deps

0 runtime dependencies. Tiny supply chain, fully typed surface.

Signals that make sense

No virtual DOM, no reconciliation. Update a signal — only what depends on it recomputes.

counter.ts
// no magic, no diffing
const count = signal(0);
const doubled = computed(() => count() * 2);
effect(() => console.log(doubled()));

count.set(21); // → logs 42
signal→ computed→ effect
