# @vanrot/runtime

Tiny reactive runtime for Vanrot: signals, mounting, and lifecycle.

Part of the [Vanrot](https://github.com/FendyHaddad/vanrot) framework.

## Installation

```sh
pnpm add @vanrot/runtime
```
