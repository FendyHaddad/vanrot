# @vanrot/vite-plugin

Vite plugin that compiles and serves Vanrot applications.

Part of the [Vanrot](https://github.com/FendyHaddad/vanrot) framework.

## Installation

```sh
pnpm add -D @vanrot/vite-plugin
```

## Usage

```ts
import { vanrot } from '@vanrot/vite-plugin';

export default {
  plugins: [vanrot()],
};
```
