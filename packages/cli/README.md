# @vanrot/cli

Vanrot command-line interface: dev server, build, doctor, and scaffolding.

Part of the [Vanrot](https://github.com/FendyHaddad/vanrot) framework.

## Installation

```sh
pnpm add -D @vanrot/cli
```

## Usage

```sh
vr dev
vr build
vr doctor
```
